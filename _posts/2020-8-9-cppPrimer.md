## 五、指针 *

解引用符：*

面对比较复杂的指针或引用的声明语句时，从右向左阅读有助于弄清楚他的真实含义

~~~c++
int a = 3;
int *b = &a;
	
a = 5;

cout << b << endl; 	//输出a的地址
cout << &a << endl; //同上
cout << &*b << endl;//同上
cout << *b << endl;	//输出值：5

//------------------------------------

*b = 2;

out << *b << endl;	//输出值：2

//------------------------------------

int *c = b;

cout << *c << endl;	//输出值：2

//------------------------------------

int *d;				//定义但是值不确定
d = c;

cout << *d << endl;	//输出值：2

//------------------------------------

c = 0;				//现在c不指向任何对象，是空指针
*d = 0;				//改变值

//--------------void*指针--------------

void *e = &a;

cout << e << endl;	//a的地址
cout << *e << endl;	//报错

int *f= (int *)e;

cout << *f << endl;	//输出值：2

//概括来说，void*指针的内存空间仅仅是内存空间，无法访问内存空间中所存对象
//void*指针能做的事比较有限：拿他和别的指针比较、作为函数的输入输出，或者赋给另一个void*指针
//因为我们不知道这个对象到底是什么类型，所以无法确定在这个对象上能做哪些操作

//------------------------------------

int* g1, g2;

cout << typeid(g1).name() << endl;	//结果为int *
cout << typeid(g2).name() << endl;	//结果为int

//------------------------------------

int h = 1;
int *hi = &h;
int **hii = &hi;

cout << *hi << endl;	//都为i的值
cout << **hii << endl;

cout << hi << endl;		//都为i的地址
cout << *hii << endl;

//------------------------------------
int *j = 2;	//报错，右值必须是对象
int *k = 0;	//被初始化，但是没有指向任何对象。不报错，但是运行空指针异常

~~~



# 六、常量

~~~c++
const int &a = 42;	//与上面的引用符相对应。合法，对常量的引用
//------------------------------------

const int b = 1;
int *c = &b;		//报错，指针不能指向常量
const int *d = &b	//合法，常量指针可以指向常量 
//------------------------------------

int d = 1;
const int *e = &d;
	
d = 2;
cout << *e << endl;	//常量最终还是被改变了，结果为2
~~~
